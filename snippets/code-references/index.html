Global References

Add references to references.csv or references.json if you want all your code snippets to use the same references. { "fapi.book": "example-references/api#book", "fapi.query": "example-references/api#query" } import * as fapi from 'flight-api' function bookFlight(flightInfo) { const confirmation = fapi.book(flightInfo) //... } function flightStatus(id) { const fullStatus = fapi.query(id) return fullStatus.shortStatus }
